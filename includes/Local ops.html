Operaciones locales

Add

Despues generar o modificar un archivo se tiene que usar el comando add.

Este comando mueve al índice las modificaciones que hayamos realizado. El índice es un snapshot del contenido del área de trabajo en un momento dado. Este snapshot es el que posteriormente se convertirá en un commit.

Commit

Despues de usar el comando add, se usa el comando commit, con esto el archivo estara incluído en el HEAD, pero aún no en tu repositorio remoto; para lograr eso se usan los comandos de push y pull.
